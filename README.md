# ATM Withdrawal Logic (Java)

This is a simple Java program that simulates an ATM withdrawal system
using conditional statements.

## Features
- Validates withdrawal amount
- Ensures amount is in multiples of 100
- Checks for sufficient balance
- Handles invalid inputs

## Concepts Used
- if-else conditions
- User input using Scanner
- Basic Java syntax

## How to Run
1. Compile the program
2. Run ATM.java
3. Enter account balance and withdrawal amount when prompted
